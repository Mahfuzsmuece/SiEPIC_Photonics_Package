In order to analyze EHVA measurement results, one needs to use the EHVA app to export a .csv file containing all the data presented by EHVA.

1. Open EHVA App
2. Use credentials in "DreamPhotonicsConfig" to connect to the databases
3. Open result manager
4. Right click in top left window (under Fields)
5. Select 'Open Query' and select 'FullDataQuery_v1' and submit
6. Run the query
7. Right click query results and select 'Raw Export'
8. Save as EHVAdata.csv
